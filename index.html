GIT AND GITHUB

In this website, I will explain simply how to use git and github

What is Git?

Git is a version control tool. It enables you to track changes that you have made to you code locally(on your computer). Once git has been installed, configure the username and useremail.

git config --global user.name " "

git config --global user.email

After the above configurations. Initialize the folder you want to track changes on using the following command

git init

Git init makes a folder a git repository. What actually happens is, a folder called .git is created and added to you current folder. Only that it is hidden. To see it, go to views then click hidden items
